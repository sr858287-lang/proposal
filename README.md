# 💌 A Special Message For You

An interactive proposal website designed to create a memorable and personalized experience. Instead of asking someone out directly, the website guides them through a series of fun questions before revealing a heartfelt proposal.

## ✨ Features

- 🎁 Beautiful multi-step interactive experience
- 🍫 Favourite chocolate input
- 🌸 Favourite flower input
- 🎨 Favourite colour selection
- 📍 Favourite date destination selection
- 🍽️ Favourite meal selection
- 📅 Date picker for choosing a preferred date
- 🎉 Celebration screen with confetti animation
- 💾 Responses saved locally using IndexedDB
- 📱 Fully responsive design for desktop and mobile

## 📖 How It Works

1. User starts the experience.
2. Answers a series of personal questions.
3. Chooses a preferred date.
4. Receives the final proposal.
5. If accepted, a celebration screen is displayed.
6. Responses are stored locally in the browser.

## ⚠️ Note

Currently, all responses are stored locally in the user's browser using IndexedDB. No data is sent to any server.

For real-time notifications (such as WhatsApp or email), backend services like Firebase can be integrated.
